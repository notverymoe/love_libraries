# About 

Welcome to Natalie's Love2D Library Repository. You can find libraries I've either written or modified here. See each library for information regarding it's purpose, usage and license. Hopefully there'll be something you'll like here.

| Library    | Description   |
|------------|---------------|
| Sardine    | A simple archetype-based ECS from scratch |
| ShripVec   | Vec2D Library (ffi/no-ffi), forked from [Brinevec](https://github.com/novemberisms/brinevector) |
| TimerFixed | Simple timer library for implementing a fixed accumulator-based timestep |
| XML        | LuaLS Typed & Relocatable version of [xml2lua's](https://github.com/manoelcampos/xml2lua) DOM XML parser |
| TiledTMX   | TMX LuaLS Typing and Parsing Implemention for Tiled Projects |
| Table2Str  | Pretty prints a lua type recusrively to a string |
